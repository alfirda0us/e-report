# E-Report

E-Report is a web-based application designed to help schools manage student grades and generate digital report cards efficiently. This project aims to simplify the reporting process by replacing manual paperwork with a modern, structured web platform.

## Features
- Add, edit, and delete student data  
- Input and manage subject scores  
- Automatically calculate final grades  
- Generate and display digital report cards  
- Responsive layout with Bootstrap  

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)  
- Bootstrap 5  

## What I Learned
Through this project, I learned how to:
- Build CRUD-based web applications  
- Manage dynamic data using JavaScript  
- Implement responsive design for various screen sizes  
- Apply clean UI/UX principles to improve usability  

## Future Improvements
- Integration with a backend database (e.g., Firebase or Node.js)  
- Secure authentication system for teachers and students  
- Export report cards as PDF  
- Admin dashboard for advanced analytics
